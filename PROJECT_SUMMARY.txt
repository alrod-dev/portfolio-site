================================================================================
HOW TO USE
================================================================================

1. INSTALLATION
   cd /sessions/jolly-great-knuth/mnt/Github-Projects/portfolio-site
   npm install

2. LOCAL DEVELOPMENT
   npm run dev
   Open http://localhost:3000

3. CUSTOMIZATION
   - Update data/index.ts with your information
   - Modify colors in tailwind.config.ts
   - Add images to public/ folder
   - Update social links in data/index.ts

4. BUILD FOR PRODUCTION
   npm run build
   npm start

5. DEPLOYMENT
   - Push to GitHub
   - Connect to Vercel
   - Auto-deploys on push
   - See SETUP.md for detailed deployment guide
